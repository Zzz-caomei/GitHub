# Code Of Conduct

## Our Standard

This project welcomes constructive participation. Contributors and users are expected to communicate respectfully, stay on topic, and focus on improving the software and documentation.

## Unacceptable Behavior

Unacceptable behavior includes harassment, personal attacks, discriminatory language, publishing private information without consent, spam, and intentionally disruptive behavior.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block users who violate this code of conduct.

## Scope

This code of conduct applies to project spaces including issues, pull requests, discussions, documentation, and related community channels.
