# Changelog

All notable changes to this project will be documented in this file.

The format follows the spirit of Keep a Changelog, and this project uses semantic version tags where practical.

## [1.0.0] - 2026-05-28

### Added

- Initial public release of `duanjujingtoufantui.skill`.
- FastAPI web workbench for uploading reference videos and images.
- Video frame extraction and contact sheet generation.
- Basic visual-change shot segmentation for video materials.
- OpenAI Responses API and OpenAI-compatible relay support.
- AI prompt generation for shot style, camera movement, lighting, rhythm, positive keywords, negative keywords, and reusable video prompts.
- Markdown report generation and knowledge-base file output.
- Internal no-login mode for trusted teams.
- Optional username/password mode with admin and project-level access control.
- Export support for task outputs.
- Example environment configuration, MIT license, and deployment notes.
